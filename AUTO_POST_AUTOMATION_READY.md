## 🚨 Troubleshooting:

### "Post created but publish failed"
**Reasons:**
- Instagram account not connected
- Instagram token expired
- Video file not ready yet
- Instagram API error

**Fix:**
1. Connect Instagram account in Accounts page
2. Try publishing manually from Posts page
3. Check error message in post details

### "No 9:16 vertical video found"
**Reason:** That video doesn't have 9:16 format in sheet

**Fix:**
- Make sure "9:16" column has a YouTube Shorts/Reel URL
- OR add Drive link with vertical video

### "Sheet has no data"
**Fix:** Make sure sheet is public (Anyone with link can view)
